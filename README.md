# Solar System Explorer v2

An interactive educational experience exploring our Solar System.

## Features

- 🪐 Interactive planet visualization
- 🎨 Beautiful solar system rendering
- 📖 Educational planet information
- 🌟 Real-time animations
- 🖱️ Click-to-explore functionality

## Quick Start

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000)

## Technology

- HTML5
- CSS3
- Vanilla JavaScript (no frameworks)
- Responsive design
